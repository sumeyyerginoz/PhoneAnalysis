# 📊 Phone Features Analysis

This project aims to analyze the technical specifications of smartphones and their relationship with pricing. The analysis provides insights into user preferences and helps evaluate product strategies of different brands.

---

## 🔍 Analysis Details

### 1. Operating System Preferences
- **Android and iOS distribution:**
  - Android devices cater to a broad range of income groups.
  - iOS devices target premium users.

**Correlation:** The operating system is directly related to device pricing and user preferences. For instance:
- iOS devices are generally positioned in the high-price segment.
- Android device prices are more sensitive to technical specifications.

### 2. RAM and Price Relationship
- Higher RAM capacity typically correlates with higher device prices.
  - **4 GB RAM:** Entry-level segment (price range: $100-$300).
  - **12 GB RAM:** Premium segment (price range: $1,500+).

**Correlation:** There is a strong positive relationship between RAM capacity and price (**correlation coefficient: 0.6-0.8**).

### 3. Front and Rear Camera Features
- **Xiaomi and Samsung:** Focus on rear camera resolution.
- **Apple:** Offers balanced performance for both front and rear cameras.
- Front camera features are a key decision factor for mid-range users.

**Correlation:** Moderate positive correlation between camera resolution and price (**correlation coefficient: 0.4-0.6**).

### 4. Battery Capacity and Price

- Charging technology is a more critical factor in premium devices.
